# AB Test Web App (Mark10)

## Run (dev)
```bash
npm run webapp:install
npm run webapp:dev
```

## Build
```bash
npm run webapp:build
```

Build output:
- `web-app/dist`

## Preview build
```bash
npm run webapp:preview
```

## Open in iOS Simulator (Safari)
1. Start preview server (`npm run webapp:preview`)
2. Open URL in Simulator:
```bash
npm run webapp:sim
```

Default URL:
- `http://127.0.0.1:4173/`
